## 一、作者投资策略与哲学

### 1. 核心策略: "高切低" (High-to-Low Switch)

作者最核心、贯穿始终的策略。即在高位品种兑现利润后,转移至低位有潜力的品种。

> "贵出如粪土,贱取如珠宝" —— 引用范蠡的交易哲学


具体操作逻辑:

- **卖高买低**: 涨幅大的品种逢高派发,利润转入低位优质品种

- **满仓满融配合**: 高切低叠加融资杠杆,急速放大牛市收益。每个周期瞄准30%+收益, 5个周期可实现10倍回报

- **坐轿理论**: 自我定位为"坐轿人"而非"抬轿人",在潜伏阶段需要眼光和耐心,兑现阶段则需要坚决

- **猎豹战法**: 不和野狗抢食,耐心等待最佳时机精准出击


典型案例:

- 2020年: 猪肉股(天邦/正邦)高位兑现 → 转向黄金股/水泥股

- 2024年Q4: 光伏设备(晶盛机电)高位兑现 → 转向金融科技(恒生电子)

- 2025年Q2-Q3: AI硬件高位兑现 → 全面转向AI+应用低位品种

- 2025年Q4-2026年: 持续配置AI+,在冲高时兑现部分利润,回调时再加仓
  
### 2. "抽离远比投入更重要"


作者反复强调的核心原则:

> "当市场一致性看多,并且暗流涌动,那么就需要及时抽离。"

- 投资者最大的敌人是贪婪,逢高兑现利润是最重要的纪律

- 锁定利润后保持现金,等待下一个机会

- 不和股票讲感情,股票只是赚钱工具

### 3. 周期轮动思维


作者深谙行业周期,从不固守单一行业:

- 2019年: 券商 → 黄金

- 2020年: 猪肉股(天邦/正邦/牧原) → 黄金股 → 水泥股 → 空仓等待

- 2021年: 水泥股 → 维生素 → 空仓

- 2024年: 光伏/半导体设备 → 金融科技(恒生电子) → 消费(美的集团)

- 2025年: AI硬件 → AI+应用(迪安诊断/昆仑万维/深信服等)

- 2026年: 继续深耕AI+应用,强调数据要素和网络安全


> "没有一个行业永远昌盛,花无百日红,适可而止,才会享受投资乐趣。"

### 4. 逆向思维与独立判断
  

> "投资千万不要和大众趋同。当大众认同,就等于聪明资金已经开始悄悄开溜。"

> "行情在分歧中不断前行,在一致之时走向破灭。"

> "涨出来的风险,跌出来的价值。"

### 5. 值博率(风险收益比)

选股的核心指标:

> "值博率才是关键。选择标的都是值博率超高的品种。"

具体含义: 在低位配置优质品种,潜在上涨空间远大于下跌风险,即高值博率。

## 二、对大盘(整体市场)的判断


### 1. 长牛定性

从2024年9月24日起,作者坚定看多A股长期牛市:

> "这是欠了一代人的牛市!"

> "重视人生最大机遇,这是财富重新分配的过程。"

> "A股万点不稀奇,未来沪指10000点才算大总攻信号。"


牛市逻辑:

- 月线/季线/年线突破近20年的3000点震荡区间

- 12万亿化债 + 个人养老金全面实施 + 社保保险入市 + 平准基金

- 横有多长竖有多高,一旦突破将形成真正意义的长牛

### 2. "慢牛"格局与顶层设计

> "资本市场顶层设计已经发生巨大改变,可持续赚钱效应与内需和投资挂钩,这定下了中国股市长期慢牛基础。"

- 决策层明确要"慢牛,不要疯牛"

- 国家安全部表态"金融安则国安"

- 本质是"赶鸭子上架行情",无形之手掌控节奏

### 3. 波浪理论定位


作者用改良的波浪理论定位市场:

- 2024.9.24 = 大一浪起点(普涨)

- 2024.12~2025.04 = 大二浪(调整)

- 2025.04清明后 = 大三浪第一子浪(主升)

- 2025.08.25后 = 大三浪第二子浪(震荡调整,高位股补跌,滞涨股补涨)

- 2026年初: 第二子浪可能面临尾声,为三浪三做准备

> "第二子浪,震荡越久,未来三浪三的力度和时间跨度就越可观。"

### 4. "无形之手"理论

作者高度重视政府对市场的调控能力,称之为"无形之手":

- 大盘有底线,无形之手不会让大盘崩溃

- 涨多了压制(拉抬银行等权重股抽血),跌多了托底(ETF买入)

- 护盘时段: 通常在11:15、13:30、14:30等关键时点操作

- 判断无形之手意图: 观察银行/保险权重股走向、510050/510300/510500/512100等ETF资金流向

### 5. 市场阶段性判断框架

- **短期谨慎乐观**: 考虑外围市场波动和情绪恢复

- **中期乐观**: 化债牛在路上,政策面和资金面宽松

- **长期超乐观**: A股相比美股仍在谷底,长期必然崛起

## 三、分析大盘和个股走势的方法论

### 1. 宏观政策面分析(最优先)

- 顶层设计 > 经济数据 > 技术面

- 密切跟踪政策动向: 中央经济工作会议、两会、重大会议公报

- 分析无形之手的行为模式和底线思维

- 关注全球宏观: 美元走势、美债收益率、美股表现(道指/纳斯达克分化)

### 2. 资金面分析

- **ETF动向**: 510050/510300/510500/512100的资金流向

- **A50期指**: 判断外资动向和A股方向的先行指标

- **两融数据**: 融资余额变化反映市场情绪

- **北向资金**: 外资进出信号

- **成交量**: "高位缩量是转弱信号","只要保持放量,啥都可能"

- **超大资金动向**: 观察主力资金的真实意图,而非散户情绪

### 3. 技术分析工具


| 指标 | 用途 |

|------|------|

| **周线MACD金叉/死叉** | 中期趋势起点/终点的核心判断 |

| **周线底背离/顶背离** | 阶段性底部和顶部信号 |

| **日线MACD死叉** | 短期趋势转弱信号 |

| **月线K线形态**(吊顶线等) | 高位见顶信号 |

| **季线金叉** | 超长期趋势信号,"珍惜第五次季线金叉" |

| **成交量分析** | 放量突破/缩量调整/地量见底 |

| **60分钟背离** | 短线操作参考 |

| **30分钟走势** | 诱多/诱空判断 |

  
### 4. 估值分析


- **市净率(PB)**: 衡量行业泡沫与低估的核心指标

> "硬件市净率长期只有软件的三分之一,现在居然超出软件一倍,蕴含的风险不言而喻。"

- **动态PE**: 结合业绩预期的估值判断

- **猪粮比**: 猪肉股特有的周期判断指标(破15需警惕,破12应离场)

### 5. 盘面语言解读


- **抵抗式下跌**: 不是见底信号,每天套牢不同资金

> "A股历史上,阶段性见底大多是以恐慌告终,最起码出现一次局部性恐慌。"

- **对倒砸盘**: 超大资金制造假象洗筹,需辨别真正的出货和洗盘

- **高标股崩溃**: 标志情绪性行情结束,理性回归

- **跷跷板效应**: 银行权重与题材股、硬件与软件之间的资金轮动

- **科创板50**: 作为最弱指数,是判断市场整体强弱的风向标


### 6. 行业比较与个股分析

- **同行业横向比较**: 对比同板块内不同个股的涨跌幅、估值和基本面

- **强弱对比**: "比较个股和板块强弱,不是看涨多高,而是看买盘是否积极"

- **主力行为分析**: 通过盘中大单交易、尾盘走势判断主力意图

- **转债与正股联动**: 通过可转债走势验证正股安全性

## 四、重点看好与看空的板块/个股(按时间演进)

### 2019-2020年: 周期股为主

- **看好**: 黄金股(山东黄金/赤峰黄金)、猪肉股(天邦/正邦/唐人神)、水泥股(海螺水泥/华新水泥)、维生素(新和成)

- **看空**: 科技股泡沫、过高估值的权重股

### 2021年: 周期见顶转防守

- **看好**: 水泥股(受益限电限产)、农业股(北大荒)

- **看空**: 高估值核心资产(杀估值将长期化)

### 2024年Q4: 牛市初期配置

- **看好**: 金融科技(恒生电子)、光伏困境反转(晶盛机电/奥特维)、消费避险(美的集团)

- **看空**: 高标股(海能达/双成药业)、微盘股

### 2025年: AI时代核心转换

- **看好(上半年→下半年演进)**:

- Q1-Q2: 仍持有光伏设备和金融科技

- Q3起全面转向**AI+应用**: 迪安诊断(AI医疗)、昆仑万维(AI大模型)、深信服(网络安全)、金山办公(AI办公)、恒生电子(金融数据)

- 补充方向: 数据要素、网络安全、横店东磁(磁性材料)、长海股份(PCB上游)、保龄宝(消费)

- **坚决看空**: AI硬件(新易盛/中际旭创/胜宏科技/沪电股份/东山精密/工业富联)、高位芯片(寒武纪/海光信息)、新能源(宁德时代/阳光电源)

### 2026年: 深耕AI+战略资产

- **核心看好**:

- **AI+应用**: 迪安诊断、昆仑万维、金山办公、中控技术(工业AI)、合合信息

- **数据要素**: AI时代数据为王,垂直类吊打通用类

- **网络安全**: 央企控股品种,未来战争的矛和盾

- **坚决看空**:

- AI硬件: "败局已定","硬件是昨日黄花"

- 传统白马: 贵州茅台/招商银行/长江电力/中国神华 → "时代翻篇"

- 白酒: "月线没止跌信号"

- 房地产: "十年内不关注"

- 智能驾驶: 安全问题未解决,不宜炒作

- 人形机器人: 太早,概念先行
