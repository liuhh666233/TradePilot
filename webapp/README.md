# Webapp Dev Starter
React + TypeScript + Vite development template

## Setup

```bash
# Initialize project
nix flake init -t "git+ssh://git@github.com/liuhh666233/MyFlake#webapp-dev-starter" --refresh

# Enter dev environment
nix develop

# Install dependencies
yarn install

# Start dev server
yarn dev

# Build for production
yarn build
```
